# Run and Deploy Your AI Studio App

This repository contains everything needed to run the app locally.

View your app in AI Studio: https://ai.studio/apps/c239d656-7dd8-462d-bdbf-5cd6411f3cf0

## Run Locally

**Prerequisite:** Node.js

1. Install dependencies:
   `npm install`
2. Set `GEMINI_API_KEY` in `.env.local` to your Gemini API key
3. Run the app:
   `npm run dev`
